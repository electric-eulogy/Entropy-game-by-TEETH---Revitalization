This is a rebuild for a game called Entropy by TEETH. This game was originally available over rpgmaker(dot)net.  Built off of old memories, screen shots, videos, and some files that I managed to find online.

This includes an archive of files I used in my rebuild of Entropy by TEETH. 

The archive is the original files that I managed to track down for my rebuild. The text is all in Russian, which I translated to English in the game that I uploaded.

I pieced together what I could from old screenshots, videos, and web archives. This build was completed in March 2024.

This project is an attempt to save this game from becoming lost media. I am sharing it in hopes that it will reach other people who remember this game.
